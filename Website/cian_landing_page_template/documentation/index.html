Twitter Feed

To show your tweets you have to follow the following steps:

  1. Visit https://dev.twitter.com/apps/ and sign in using your Twitter username and password. This doesn't have to be the username or password of the stream you need access to, just a Twitter account you control.
  2. Select 'Create new app' and enter the application details.
    1. The name and description can be anything you like really, but you can't use 'Twitter' in the name.
    2. The website field can be your main website and doesn't have to be the site where your Twitter feed or feeds are located.
    3. Callback URL can be left blank.
  3. When the data are completed, click to the button 'Create your Twitter application'
  4. Enter the CAPTCHA info and click create.
  5. On the next details screen, click in the tab 'API keys'.
  6. In this tab, you have to generate the 'access token'. Click the button 'Create my access token'. If the first time doesn't work, try it again.
  7. Make note of the Consumer key, Consumer secret, Access token and Access token secret as highlighted below.
  8. Change the next variables into the file php/get-tweets.php:

    $consumerkey = "12345";
    $consumersecret = "12345";
    $accesstoken = "12345";
    $accesstokensecret = "12345";
    ...
  9. Finally, we can change the number of tweets and the twitter profiles in the file js/twitterfeed.js:
